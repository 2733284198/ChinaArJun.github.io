## 名字由来

大学时读了温庭筠一首词，`不知心底事，水风空落眼前花`，于是就起了一个名字: 水风。

后来难免觉得花间词过于矫情，于是把名字拆成两个，只取前两个字：。取李白 `暮从碧山下，随人归` 之意，恰好那段时间从京西阳台山下来时伴着月亮。

再后来读书读多了，觉得 `松风吹解带，照弹琴` 与 `水风轻，蘋花渐老，月露冷，梧叶飘黄` 的意境也不错，也很喜欢。所以我两台服务器的 hostname 被命名为这两个名字。
